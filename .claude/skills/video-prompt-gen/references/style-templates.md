## 1. Pixel Art

### Style Keywords

```
pixel art style, 16-bit aesthetic, limited color palette, retro game graphics,
crisp pixel edges, no anti-aliasing, SNES era, NES style, game sprite
```

### Resolution Variants

| Style | Keywords |
|-------|----------|
| 8-bit | `8-bit style, NES aesthetic, very limited palette, chunky pixels` |
| 16-bit | `16-bit style, SNES aesthetic, 64 color palette, detailed sprites` |
| 32-bit | `32-bit style, PlayStation 1 era, pre-rendered sprites, more colors` |

### Color Palettes

| Theme | Keywords |
|-------|----------|
| Classic | `vibrant primary colors, high contrast` |
| Muted | `desaturated earth tones, muted palette` |
| Neon | `cyberpunk neon colors, glowing effects, dark background` |
| Pastel | `soft pastel colors, gentle aesthetic` |

### Full Template Example

**Idle Animation:**
```
Static camera, side view, full shot. A [character description] in pixel art
style. The character stands in a relaxed idle pose with subtle breathing
animation and occasional small movements. Solid green background. 16-bit
pixel art style, SNES game aesthetic, limited color palette with [color
theme], crisp pixel edges, no anti-aliasing, retro game sprite, 320x240
resolution feel.
```

**Negative Prompt:**
```
anti-aliasing, smooth gradients, photorealistic, blur effects, complex
background, detailed scenery, high resolution textures
```

## 2. 2D Cartoon

### Style Keywords

```
2D cartoon style, flat colors, bold outlines, animated series look,
vector art style, clean lines, cel-shaded
```

### Sub-variants

| Style | Keywords |
|-------|----------|
| Flat | `flat design, minimal shading, geometric shapes, modern cartoon` |
| Classic | `classic animation style, hand-drawn look, traditional 2D animation` |
| Modern | `modern cartoon network style, expressive, dynamic poses` |
| Chibi | `chibi style, big head small body, cute proportions, super deformed` |

### Line Styles

| Style | Keywords |
|-------|----------|
| Thick | `thick bold outlines, graphic novel style` |
| Thin | `thin delicate lines, fine detail` |
| Sketchy | `sketchy hand-drawn lines, organic feel` |
| Clean | `clean vector lines, digital perfection` |

### Full Template Example

**Walk Animation:**
```
Static camera, side view, full shot. A [character description] in 2D cartoon
style. The character walks at a steady pace with a natural bouncy gait,
showing personality through their walking style. Solid white background.
2D cartoon style, flat colors with subtle cel-shading, thick bold outlines,
animated series quality, expressive character design, clean vector look.
```

**Negative Prompt:**
```
realistic proportions, photorealistic, complex shading, gradient fills,
3D rendered, detailed textures
```

## 4. Realistic

```
photorealistic, detailed textures, natural lighting, cinematic quality,

```

### Lighting Variants

| Style | Keywords |
|-------|----------|
| Studio | `studio lighting, soft box lighting, professional photography` |
| Natural | `natural daylight, soft shadows, realistic sun lighting` |
| Dramatic | `dramatic rim lighting, strong contrast, cinematic lighting` |
| Indoor | `indoor lighting, warm artificial light, ambient occlusion` |

### Quality Levels

| Level | Keywords |
|-------|----------|
| High | `8K resolution, ultra detailed, photorealistic rendering` |
| Medium | `4K resolution, detailed, realistic` |
| Stylized | `stylized realism, slight artistic interpretation` |

### Full Template Example

**Run Animation:**
```
Static camera, side view, full shot. A [character description] in
photorealistic style. The character runs at full speed with athletic form,
showing muscle tension, cloth physics, and hair movement. Natural motion
with proper weight and momentum. Solid green background. Photorealistic
```

**Negative Prompt:**
```
cartoon, anime, stylized, low resolution, blur, noise, artificial look,
```

## Quick Reference: Action Prompts

### Idle

```
standing in neutral pose, subtle breathing animation, gentle weight shifts,
occasional blink, relaxed posture, small idle movements
```

### Walk

```
walking cycle, steady natural pace, full stride, arm swing, natural gait,
consistent speed, loop-friendly start and end pose
```

### Run

```
running cycle, fast athletic pace, dynamic stride, forward lean, pumping
arms, determined expression, high energy movement
```

### Jump

```
crouch anticipation, powerful upward leap, peak pose with extended body,
landing preparation, smooth arc trajectory, dynamic silhouette
```

## Quick Reference: Background Keywords

### Best for Background Removal

| Type | Keywords | Rembg Quality |
|------|----------|---------------|
| Green | `solid green background` | Excellent |
| White | `plain white background` | Excellent |
| Black | `pure black background` | Very Good |
| Blue | `solid blue background` | Very Good |
| Gray | `neutral gray background` | Good |

### Avoid (Difficult for Extraction)

- Complex outdoor scenes
- Multiple objects in background
- Gradient backgrounds with similar colors to character
- Patterned backgrounds
- Realistic environments

## Character Description Template

When building character descriptions, use this structure:

```
[Role/Type] with [Physical Features], wearing [Clothing/Armor],
[Accessories/Weapons], [Color Scheme]
```

**Examples:**

- `warrior knight with broad shoulders and scarred face, wearing silver
  plate armor with gold trim, holding a large sword, blue and gold color
  scheme`

- `mage character with long white hair and glowing blue eyes, wearing
  dark blue robes with silver embroidery, holding a crystal staff,
  purple and silver color scheme`

- `rogue assassin with athletic build and sharp features, wearing black
  leather armor with hood, dual daggers at belt, black and red color
  scheme`

- `archer with slim build and determined expression, wearing green ranger
  cloak and leather armor, carrying a longbow, green and brown color
  scheme`
