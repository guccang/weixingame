# 图片裁切工具

根据行列数自动裁切图片为多个独立的图标，输出 PNG 格式。

## 使用方法

```
/img-split <图片路径> <行数> <列数> [输出目录]
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `图片路径` | 输入图片路径（必需） |
| `行数` | 裁切行数（必需） |
| `列数` | 裁切列数（必需） |
| `输出目录` | 输出目录（可选，默认在输入文件同目录创建 `_icons` 文件夹） |

### 示例

```
# 裁切为 2行2列
/img-split icons.png 2 2

# 裁切为 3行4列
/img-split sprite.png 3 4

# 指定输出目录
/img-split icons.png 2 2 ./my_icons
```

## 输出文件命名

裁切后的图标按行列位置命名：`icon_<行号>_<列号>.png`

例如 2x2 的裁切结果：
```
icon_0_0.png  icon_0_1.png
icon_1_0.png  icon_1_1.png
```

## 注意事项

- 确保输入图片尺寸能被行列数整除
- 如果不能整除，边缘可能会有少量像素损失
- 支持透明背景的 PNG 图片
