## 说明
游戏数据表编辑工具，用于管理 `jump-tower-game/tables/` 目录下的 `.txt` 表格文件。

## 表格路径
`D:\GitWechatGame\jump-tower-game\tables\`

## 表格格式规范
表格使用制表符(Tab)分隔，格式如下：

```
第1行: 类型定义    STRING  STRING
第2行: 中文列名    #名称    资源路径
第3行: 英文列名    #Name    ResPath
第4行起: 数据行    1       糖果平台    images/xxx.png
```

**注意**: `#` 号只出现在中文列名和英文列名行的**第一列**，用于标识表头行，其他列不需要 `#`。

### 支持的数据类型
- `INT` - 整数
- `STRING` - 字符串
- `FLOAT` - 浮点数
- `BOOL` - 布尔值 (true/false)

## 操作类型

### 1. 新建表格
创建新的表格文件，需要指定：
- 表格名称（英文，如 `Items.txt`）
- 列定义（类型、中文名、英文名）
- 初始数据（可选）

### 2. 添加数据行
向现有表格添加新行，自动生成递增ID。

### 3. 修改数据
修改指定ID行的某列数据。

### 4. 删除数据行
删除指定ID的数据行，并重新排序ID。

### 5. 添加列
在现有表格中添加新列，需要指定：
- 列类型
- 中文列名
- 英文列名
- 默认值（可选）

### 6. 查询数据
查询表格内容，支持按条件筛选。

## 操作步骤

1. **确认操作类型**: 新建/添加行/修改/删除/添加列/查询
2. **读取现有表格**: 了解当前数据结构（修改/添加/删除时）
3. **执行操作**: 按规范编辑文件
4. **验证结果**: 确保格式正确
5. **执行转换脚本**: 修改表格后必须执行 Python 转换脚本生成 JS 文件
   ```bash
   python tools/table_converter.py
   ```
   此脚本会生成以下文件到 `jump-tower-game/js/tables/` 目录：
   - `tableConfig.js` - 表格元数据配置
   - `tableStruct.js` - 数据结构类定义
   - `tableData.js` - 初始数据
   - `tableManager.js` - 表格管理器


## 注意事项
- ID列始终为第一列，类型为INT，自动递增
- 列名使用驼峰命名法（英文）和中文（中文列名）
- 资源路径使用相对于 `jump-tower-game/` 的相对路径
- 修改后需保持制表符分隔格式
- 删除行后需重新排序ID保持连续
- **重要**: 表格修改后必须运行 `python tools/table_converter.py` 生成 JS 文件，否则游戏无法读取最新数据
